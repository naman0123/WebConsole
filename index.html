Visual Studio, more specifically Visual Studio Code, is
a text editor produced by Microsoft.
It's actually a misunderstanding that Visual Studio
Code is a fully functional IDE. It's more like a light fancy text editor but with extensions which will you do coding.
People also refer to it as the lighter version of the original
"Visual Studio" which is an IDE used to code Interpreted languages like Python, Javascript etc. and Compiled languages such a C++, C# etc.
To set up Visual Studio Code you can download it from there Website.


Console is a screen where the output of all code being executing is printed.
We use console for debugging and making sure if everything is being executed correctly.
There are many ways to open console but the most common way is to
open the "Debug Console" tab in VS Code, execute you're code and you will see the output of the code.
To print anything on console use "console.log("You're output goes here")" for JavaScript and in the round brackets write your output.
